# Create Modpack

Modpack de Create y addons para Minecraft 1.21.1 NeoForge.

## Instalación

### CurseForge Launcher / Prism / MultiMC
1. Descarga `CreateModpack-CF.zip`
2. Importa el ZIP en tu launcher

### Modrinth Launcher / Prism
1. Descarga `CreateModpack.mrpack`
2. Importa el archivo `.mrpack`

### Manual
1. Instala NeoForge 1.21.1
2. Copia todos los JARs de `overrides/mods/` a `mods/`
3. Además instala los mods listados en `manifest.json` (pestaña "Files")

## Incluye

- Create + 20 addons
- CC: Tweaked + Sable
- KubeJS + KubeJS Create
- Optimización: Sodium, Lithium, FerriteCore, ModernFix, etc.
- Mejoras de calidad de vida: JEI, Jade, AppleSkin, Xaero's Map, etc.
